# Examples

- **`vibe-budgeting.snippet.yml`** — copy the `vibe-budgeting` service block into your own `docker-compose.yml`. Adjust `build.context` (path to this repo), ports, and networking (`network_mode`, Tailscale, reverse proxy, etc.) to match your setup.

The canonical minimal compose for cloning this repo alone remains **`docker-compose.yml`** at the repository root.
